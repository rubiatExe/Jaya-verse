**The Jaya Station** is a full-stack wellness and connection hub built as a gift for a loved one in a long-distance relationship. The platform is designed to bridge distance by fostering a sense of connection with friends and family, while also providing a suite of personalized tools to support daily self-care and mental well-being.

## üöÄ Core Features

This application combines real-time data synchronization, generative AI, and a gamified interface to create a uniquely personal and interactive experience.

### Connection & Community
*   **Friend Map:** An interactive world map that visualizes the locations of friends and family, showing their local time and personal messages to make them feel closer, no matter the distance.
*   **Reasons We Love You Jar:** A collaborative "jar" where friends can submit reasons they love Jaya. An AI-powered feature then randomly selects and displays a reason, delivering a surprise moment of joy.
*   **"Open When..." Letters:** A digital take on the classic concept. Friends can submit private notes for specific moods (e.g., "Open when you're feeling sad"), providing comfort and support exactly when it's needed.
*   **Live Activity Feed:** A public feed on the landing page that shows recent messages and reasons left by friends, creating a sense of a living, breathing community.

### Wellness & Self-Care
*   **Life RPG (Role-Playing Game):** A gamified status widget where Jaya can set her current activity (e.g., Reading, Creating, Traveling). Her status is updated in real-time for all visitors, turning daily life into a fun, shared experience.
*   **Self-Talk Sesh:** An AI coaching tool powered by Genkit. Users can input a self-doubting thought, and the AI reframes it into a kind, positive, and empowering perspective.
*   **Daily Trackers:** Simple, beautiful trackers for daily water intake and menstrual cycles, complete with visual rewards (like a blooming rose) for meeting goals.
*   **Dream Board:** A visual, collage-style board for personal goals and aspirations, allowing users to upload images and titles to visualize their dreams.

## üõ†Ô∏è Technology Stack

This project was built with a modern, full-stack, and cloud-native architecture.

*   **Frontend:**
    *   **Framework:** [Next.js](https://nextjs.org/) (with App Router)
    *   **Language:** [TypeScript](https://www.typescriptlang.org/)
    *   **UI Library:** [React](https://reactjs.org/)
    *   **Styling:** [Tailwind CSS](https://tailwindcss.com/)
    *   **Components:** [ShadCN/UI](https://ui.shadcn.com/) for beautiful, accessible components.

*   **Backend & Cloud:**
    *   **Database:** [Firebase Firestore](https://firebase.google.com/docs/firestore) for real-time, NoSQL data storage and synchronization.
    *   **Platform:** Deployed on **Google Cloud**.
    *   **Generative AI:** [Google's Genkit](https://firebase.google.com/docs/genkit) for creating and managing server-side AI flows.

## ‚öôÔ∏è Getting Started

To run this project locally, you will need to have Node.js and npm installed.

1.  **Clone the repository:**
    ```bash
    git clone https://github.com/rubiatExe/Jaya-verse.git
    cd Jaya-verse
    ```

2.  **Install dependencies:**
    ```bash
    npm install
    ```

3.  **Set up Environment Variables:**
    This project requires a Firebase configuration. You will need to create a `.env` file in the root of the project and add your Firebase project credentials.

4.  **Run the development server:**
    ```bash
    npm run dev
    ```

Open [http://localhost:9002](http://localhost:9002) with your browser to see the result.
